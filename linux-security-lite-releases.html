Linux Security Lite v1.2.2

Maintenance and hardening release. Bundled updater, safe upgrade preservation, and release-validation CI gate.

Published May 2026. Customer documentation baseline updated 2026-05-19.

  • ✓Bundled updater script: scripts/toolkit-updater.sh is now included in all release package formats (.deb, .rpm, .apk, tarball) from this version onwards.
  • ✓Upgrade preservation: /etc/audit-toolkit/ (config) and /var/log/audit-toolkit/ (reports) are preserved automatically during package upgrades — no manual backup required for minor upgrades.
  • ✓Safe rollback: rollback now uses rsync --delete rather than a destructive rm -rf; existing configuration and log files are not removed during a rollback operation.
  • ✓Release gate: customer-facing documentation must pass validate-customer-docs-release-version.py before a release tag is created; the pipeline blocks on version mismatch.
  • ✓Documentation set updated to version 1.2.2 (24 documents) covering installation, maintenance, licensing, and enterprise compliance platform guidance.
  • ✓OWASP security scorecard maintained at 100/100, Grade A+.
