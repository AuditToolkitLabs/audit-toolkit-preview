## Release Bundles

Customer release bundles receive the authoritative customer-facing documents
from this repository through the release-bundle documentation sync process.
See [Release Bundle Documentation Sync](../release-bundles/README.md).

Rendered website content must originate from AuditToolkit-Docs via controlled
sync. Presentation repositories must not author canonical customer
documentation content.
