Standalone Host Security — No Platform Required

A separate family of self-contained tools for single-host security auditing. Install on any machine, activate with a licence key, run an audit, and review structured findings locally. No central server. No agents. No network dependency at run time.

Sold separately from the Platform — one licence (£4,999/year) covers all three host tools.

Windows Security Lite

Read-only security checks on a single Windows host. MSI installer for Windows 10, 11, and Server 2019+. CLI and optional WinForms GUI launcher included.

Deployment Model

Built for environments where data cannot leave

Deploy every component on your own infrastructure without SaaS dependencies.

Fully self-hosted

Every component of the suite deploys on your own infrastructure. No cloud dependency. No data transmitted to external services. No SaaS lock-in. Your audit data, findings, and evidence stay in your environment.

Air-gap capable

Offline MSI installers available for restricted environments. Agent deployment supported without outbound internet access. Designed for financial services, healthcare, government, and defence supply chain environments.

BSL converts to MIT

All BSL-licensed products in this suite convert to the MIT licence by 2031. Your deployment rights are permanent regardless of any future commercial relationship. Source code becomes fully open and your investment is protected.

Telemetry is disabled by default across all products. When optionally enabled, only non-sensitive operational metadata is transmitted and no inventory, host, vulnerability, or credential data leaves your environment.
