CMDB API Data Collection Tool

Product Details → Latest GA — v0.2.6

Two-dimensional licensing model. Every commercial licence combines two independent selections: a Deployment Tier (Dimension A), which controls how many Workspaces your Installation may host, and a Plan Tier (Dimension B), which sets the per-Workspace operational limits (host cap, API rate, and scan scheduling). You must select one from each dimension.

Stripe checkout is live. Plan Tier (Dimension B) subscriptions can be purchased directly via Stripe below. Your Keygen licence key is generated and emailed automatically on successful payment. Deployment Tier (Dimension A) selection is confirmed at checkout — contact us if you need Workgroup, MSP, or OEM terms applied to your order.
Dimension A — Deployment Tier

Configured via the LICENSE_TIER environment variable at installation. Controls the maximum number of Workspaces per Installation and permitted use cases.
