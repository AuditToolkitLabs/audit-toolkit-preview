Security

Responsible Disclosure & Security Policy

We take security reports seriously. If you discover a vulnerability in any AuditToolkit Labs product, please report it privately before public disclosure.

Please do not report security vulnerabilities via public issue trackers. Use the dedicated email address below for all security disclosures.
  • 📧Email: support@audittoolkitlabs.com
    Subject: Security Vulnerability Report
  • 🕐Response time: We aim to acknowledge all security reports within 48 hours and provide an initial assessment within 5 business days.
  • 🔒Coordinated disclosure: We request a minimum 90-day coordinated disclosure window to develop and release a fix before public disclosure.
  • ✅Safe harbour: Good-faith security researchers who follow responsible disclosure will not face legal action from AuditToolkit Labs for their research activities.

FAQ

Common Questions

Is the CMDB tool source code available?

The repository is private under a Business Source Licence (BSL). Source access is provided to licensees. Documentation and release artefacts are included with each deployment release.

Can I request an evaluation deployment?

Yes. Contact us via the support email to request a technical evaluation or trial licence. We will work with you to arrange suitable access.

What deployment platforms are supported?

Linux (Docker / systemd), Windows Server (IIS / NSSM service), and hybrid deployment with the managed agent for isolated network segments.

Where is the documentation hosted?

All 34 customer documents are bundled with each release package. A full catalogue is available on the Documentation page. Documents are not individually hosted publicly due to the private repository status.

How do I report a security vulnerability?

Email support@audittoolkitlabs.com with subject "Security Vulnerability Report". We follow a coordinated 90-day disclosure policy.

What is the licence model?

AuditToolkit Labs products use the Business Source Licence (BSL). This allows internal use and evaluation but restricts commercial resale or redistribution. Contact us for full licence terms.
