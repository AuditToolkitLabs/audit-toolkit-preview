AuditToolkit Lite

Self-contained single-host security auditing for Linux, Windows, and macOS. No platform, no agents, no central server. Install on any host, activate with a licence key, run an audit, and review structured findings — entirely on-device.

Two Tiers

Enterprise Suite vs Lite Toolkit

The enterprise suite is a fleet-wide compliance platform requiring a central server and agents. The Lite Toolkit is a completely separate family — each tool runs independently on a single host with no infrastructure dependency.

Enterprise Suite
ScopeFleet-wide — hundreds of hosts
RequiresCentral platform + agents + API
DeploymentServer installation, agent rollout
OutputPlatform-held findings, evidence queue

Lite Toolkit
ScopeSingle host — one machine at a time
RequiresPython 3.10 or MSI — nothing else

OutputLocal JSON, CSV, and HTML report
Use caseHost posture check, consultant audit, pre-deployment baseline
ProductsLinux Host Lite, Windows Security Lite, macOS Security Lite

Three Platforms

One consistent audit model across Linux, Windows, and macOS

Each platform variant shares the same philosophy — read-only checks, structured findings, a Hardening Index, and local-only output — delivered through the native installer and runtime for that operating system.

Consistent Across All Platforms

Same outputs. Same licence model. Same audit philosophy.

Every Lite tool produces the same structured evidence artifacts regardless of platform. Results are immediately usable for compliance review, governance reporting, or downstream tooling.

Hardening Index

An overall posture score with per-category breakdowns so you know exactly where risk is concentrated.

audit-result.json

Full findings, scores, baseline references, and coverage metadata in a structured JSON artifact.

findings.csv

Flat export for spreadsheet review, dashboard import, or programmatic processing. (Linux and Windows.)

summary.html

Human-readable report with category scores and findings — open in any browser, no server required.

Air-gap capable

Signed offline licence files support environments with no outbound internet access. No phone-home required after activation.

Read-only checks

No system changes made during an audit run. Safe to run on production hosts without change-control overhead.

Architecture

Everything stays on the host

No central platform. No agent registration. No network dependency during audit execution. The tool runs, writes its output to the configured local path, and exits.
