Decision Guide

Which licence do I need?

Use case Licence path Action
Internal IT or security team, own environment only Community (free) or paid tier for your server count Download or request evaluation access
CI/CD pipeline integration on a single Linux host Linux Security Lite Community - free, no key required Request evaluation access
Multi-server fleet, internal use only Paid tier matching your server count Request pricing
Managed service - operating on behalf of customers Commercial MSP agreement required Contact licensing team
Embedding in a commercial product OEM agreement required before use Contact licensing team
SaaS or hosted service offering Commercial agreement required Contact licensing team

If your use case is not listed here, contact us before deployment. Using the software outside permitted scope is a material breach of the BSL terms.

MSP, MSSP, and OEM licensing

If you intend to manage customer environments, offer this suite as part of a managed security service, or embed any component in a commercial product, a separate commercial agreement is required before production use.

MSP and MSSP arrangements are supported across the suite. Commercial terms cover per-customer or per-endpoint usage reporting, quarterly royalty billing, and white-label rights where applicable.
