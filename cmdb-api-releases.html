v0.2.6 Latest GA ✓ OWASP PASS 29 April 2026

Security Verified — Enterprise Ready

Full OWASP Top 10:2021 audit completed. All findings addressed. Zero CVEs across 25 pinned production dependencies at release date.

  • OWASP Top 10:2021 full source-code audit — 8 PASS, 1 PARTIAL, 0 FAIL, zero open findings at GA
  • HTTP security headers on all responses: Content-Security-Policy, X-Frame-Options, Referrer-Policy, Permissions-Policy, X-Content-Type-Options
  • AES-256 Fernet field encryption for all connector credentials and SSO secrets at rest
  • Seed admin account now enforces must_change_password on first login (A07 remediation)
  • HAProxy connector: configurable verify_ssl field replacing hardcoded verify=False (A05 remediation SEC-003)
  • pip-audit clean — zero CVEs across all production dependencies
  • Bandit SAST scan: 0 High, 0 Medium, 0 Low findings post-remediation
  • Customer documentation: OWASP Scorecard (Doc 34) and Procurement Security FAQ (Doc 33) added
  • 542 automated tests passing; SQLAlchemy LegacyAPIWarning resolved
