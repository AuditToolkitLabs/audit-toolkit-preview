Licensing & Pricing

CMDB API Data Collection Tool

Two-dimensional licensing: choose a Deployment Tier (how many Workspaces) and a Plan Tier (per-Workspace operational limits). Contact the Licence Team to arrange your licence.

Request a Quote → Product Details
Licensing is currently handled manually. Email License@audittoolkitlabs.com to request a licence — include your required Deployment Tier and Plan Tier. We will issue your Keygen key directly. Contact us separately for Workgroup, MSP, or OEM terms.
Dimension A — Deployment Tier

Controls the maximum number of Workspaces per Installation and permitted use cases. Configured via the LICENSE_TIER environment variable.

Standard

Standard

Single Workspace per Installation. For a single organisation managing its own internal IT assets.

  • 1 Workspace per Installation
  • Single legal entity — own IT use only
  • One licence per Installation
  • Workspace partitioning by dept / BU / environment
  • Managed Customer Workspaces

Select a Plan Tier below, then contact us to confirm Standard deployment terms.

MSP

MSP

Unlimited Workspaces per Installation. For Managed Service Providers operating the platform for multiple end customers. Per-endpoint royalty applies.

  • Unlimited Workspaces per Installation
  • Each Workspace = one Managed Customer
  • Per-endpoint royalty — quarterly usage report
  • White-labelling / re-branding (requires OEM)

Subscription + per-endpoint quarterly royalty.

OEM

OEM / Embedded

Permits redistribution, embedding, white-labelling, and all commercial uses not covered by other tiers. Requires a separately negotiated written agreement.

  • Unlimited Workspaces per Installation
  • Redistribution and embedding permitted
  • White-labelling and re-branding permitted
  • All commercial use cases

Requires a separately signed written agreement. Setting LICENSE_TIER=oem without such an agreement is a material breach.
