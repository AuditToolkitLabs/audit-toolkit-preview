Audit Admin Toolkit - 1,019 audit scripts. 712 fix scripts. Atomic rollback. Auditor-ready evidence.

Self-hosted audit and remediation platform for Linux, Windows, and hypervisor estates with controlled execution, rollback safety, and evidence automation.

Features

Coverage, remediation, discovery, and evidence in one toolkit

The platform combines large script coverage, controlled change execution, and evidence output aligned for auditors.

Script coverage

  • ✓1,019 audit scripts - Linux (500), Windows (503), hypervisors (16).
  • ✓712 runnable fix scripts - Linux (348), Windows (353), hypervisors (11).
  • ✓Linux domains: platform, cloud, apps, network, security, storage, web, advanced controls, automation, data, Alpine.
  • ✓Windows domains: common, apps, desktop, server, platform.
  • ✓Hypervisor coverage: ESXi, KVM, Nutanix, Proxmox, Xen.
View full coverage matrix

Remediation

  • ✓Transaction-safe atomic remediation: backup, apply, validate, rollback in one transaction.
  • ✓Dead man's switch with automatic rollback if confirmation is not received in timeout window.
  • ✓SuperAdmin-gated execution with required secondary auth token.
  • ✓Dry-run preview before every execution.
  • ✓Rollback by execution ID over SSH (Linux) and WinRM (Windows).
View remediation model

Asset discovery

  • ✓Integrated asset discovery with no separate tool required for baseline inventory.
  • ✓WinRM-based Windows and SSH-based Linux collection.
  • ✓CVE and KEV correlation per discovered software version.
  • ✓Continuous or scheduled discovery jobs.

Evidence and compliance

  • ✓SHA-256 checksums on every artifact.
  • ✓MANIFEST.json and SHA256SUMS.txt in every package.
  • ✓Framework mapping to NIST SP 800-53, CIS, and PCI-DSS references.
  • ✓Auditor-ready ZIP export with digital signatures for chain-of-custody.
  • ✓30/60/90 day compliance trajectory trends.

Integrations

  • ✓SIEM: Splunk, Elastic, Sentinel, Wazuh, QRadar, Graylog, Prometheus/Grafana.
  • ✓Ticketing: ServiceNow and Jira with automated ticket workflows.
  • ✓Chat: Slack and Microsoft Teams for digest and alerts.
  • ✓CI/CD: GitHub Actions with SARIF, PR comments, and status checks.
