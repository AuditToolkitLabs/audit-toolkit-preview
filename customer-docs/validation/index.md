# Validation And Release Gates

This section is the central authority for release readiness, security
scorecards, packaging evidence, and the gates a build must pass before it is
customer-ready. Product repositories keep only the local command used to run a
validation; final reports and gates live here.

## Pages

- [Release Gates](release-gates.md)
- [Offline Fallback - First Live Issuance Validation Summary](offline-fallback/first-live-issuance-validation-summary.md)

## Planned Central Records

- Security scorecards (OWASP) per product and release
- STIG and CIS coverage matrices
- Packaging evidence (DEB/RPM/MSI payload verification)
- Readiness reports per product and release
