# Integration Quick Starts

## Overview

These quick starts cover the most common ways customers begin using Switch
Exposure Center after installation.

## Suggested order

1. Configure inventory connectors.
2. Configure advisory sources.
3. Run a manual refresh.
4. Review the inventory and exposure views.
5. Schedule recurring refreshes.

## Available quick starts

- Advisory refresh and correlation
- API usage for common lookups
- External scheduler setup for recurring checks

## When to use quick starts

Use quick starts when a team wants a short, task-oriented guide instead of the
full administration or operations documentation.
