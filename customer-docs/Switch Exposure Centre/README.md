# Switch Exposure Center Customer Documentation

This folder contains customer-facing documentation for Switch Exposure Center.
It is written to support operators, admins, support teams, and technical
stakeholders who need to understand the product, its workflows, and the
operational boundaries around it.

The set is intentionally structured like a small documentation pack:
core product guides first, then operational and security material, then quick
starts and appendices.

Start with [index.md](index.md) for the reading order.

For customer due-diligence and procurement review, start with the
[Security and Quality Assurance Report](26-security-and-quality-assurance-report.md).
