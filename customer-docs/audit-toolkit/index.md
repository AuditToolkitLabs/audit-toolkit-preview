## How to use this documentation

This documentation describes what to expect when using, administering
and operating the Security Audit Toolkit deployed within your
environment. It is written for:

- IT and security operations staff
- Application administrators
- Compliance and assurance teams
- End users performing or reviewing security audits
- Auditors and assessors

Each section maps to one or more ISO/IEC 20000-1:2018 clauses, shown in
italics next to the section title.
