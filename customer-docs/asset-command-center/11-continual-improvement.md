# 11. Continual Improvement

Customers should review Asset Command Centre usage and operating policy
regularly.

## Recommended review cadence

- monthly: user and privileged-role review
- monthly: connector allowlist and discovery coverage review
- quarterly: MFA and credential-rotation review
- quarterly: backup and restore test
- quarterly: support backlog and release adoption review

Feedback and defect reports for this product should be routed through
[Support@audittoolkitlabs.com](mailto:Support@audittoolkitlabs.com) or
[Security@audittoolkitlabs.com](mailto:Security@audittoolkitlabs.com)
for security issues.
