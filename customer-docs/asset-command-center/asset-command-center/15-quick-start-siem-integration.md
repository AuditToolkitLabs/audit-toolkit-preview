# 15. SIEM and Security Webhook Notes

The Admin UI includes SIEM and notification configuration fields, and
security-event webhook behavior may also be supplied by deployment
settings.

## Supported documentation position

This documentation set describes those surfaces generically and does not
claim release-ready, vendor-specific setup guides for named SIEM
products.

## Customer guidance

- validate the target destination in the customer environment
- confirm retention and transport security outside the product
- test event delivery after any settings change
- treat workspace, tenant, and endpoint identifiers as customer-owned
  integration parameters
