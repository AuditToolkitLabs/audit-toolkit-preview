# 10. Monitoring and Reporting

The product provides local status and reporting views for operators and
administrators.

## Key monitoring surfaces

- Overview page for estate and connector summary
- Task Runs page for collection-job execution state
- Reporting page for local exports and trend views
- Admin readiness and settings views for privileged health checks
- Logs and portal views for super-admin troubleshooting

## Reporting position

The documented reporting model for this release is local reporting and
export from the standalone node, with optional upstream forwarding when
a central deployment is enabled.
