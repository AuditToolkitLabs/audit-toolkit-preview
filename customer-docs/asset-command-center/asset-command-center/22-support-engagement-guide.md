# 22. Support Engagement Guide

When contacting AuditToolkit Labs, describe the issue as an Asset
Command Center issue and include the release version, deployment model,
and the user-visible symptom.

## Include in support requests

- product version (planned release baseline for this cycle: 1.0.2)
- deployment type
- host OS and runtime summary
- reproduction steps
- relevant screenshots or exports
- relevant log excerpts where permitted
