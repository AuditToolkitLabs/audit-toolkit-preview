# 7. Operations and Support Model

Asset Command Center is customer-operated software. AuditToolkit Labs
provides the product, documentation, and commercial support associated
with the licensed release.

## Responsibility split

| Area | Customer | AuditToolkit Labs |
| --- | --- | --- |
| Host OS and network operations | Yes | No |
| Database operations and backups | Yes | No |
| Product defects and release packages | No | Yes |
| License issuance and entitlement changes | No | Yes |
| Day-to-day user and connector administration | Yes | No |

## Licensing purchase channel

All pricing, purchase, and upgrade actions are managed at
[https://audittoolkitlabs.com/](https://audittoolkitlabs.com/). The deployed
product is intentionally limited to license key entry and validation workflows.
