# Operator Runbook

Status: Authoritative
Date: 2026-06-12

This operator runbook is maintained in AuditToolkit-Docs as the canonical
operator reference source.

Presentation repositories must render this content through controlled sync and
must not maintain independently authored operator-runbook variants.

## Runbook Scope

- service lifecycle controls
- operational checks and health verification
- controlled maintenance and upgrade execution
- incident triage and escalation handoff
- support evidence capture boundaries

This document is an authoritative source artifact. Customer release bundles may
exclude operator material where policy requires, but ownership remains central.
