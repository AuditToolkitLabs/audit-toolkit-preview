# Licensing Authority

This page defines the licensing authority boundary for AuditToolkit
documentation.

## Canonical Source

The `licensing/` section in AuditToolkit-Docs is the canonical licensing
authority for:

- Tier definitions
- License mode behavior
- Entitlement interpretation
- Licensing drift markers and gating terminology

Product repositories must keep only product-specific operational instructions
(paths, env vars, UI flow notes) and link to central licensing authority.

## Canonical Licensing Documents

- [Licensing Overview](overview.md)

## Repository Rule

When local product docs mention licensing policy, they must defer authority to
central licensing pages in this section.
