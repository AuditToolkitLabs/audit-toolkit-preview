# AuditToolkit Platform Administrator Guide

## Introduction

Welcome to the comprehensive Administrator Guide for the AuditToolkit platform. This guide is the central resource for system administrators responsible for deploying, configuring, managing, and securing the AuditToolkit suite of products.

Our goal is to provide clear, step-by-step instructions to ensure a smooth and secure operational experience. This document consolidates critical administrative information and links to detailed, product-specific documentation where necessary.

## Core Administrative Topics

This guide is organized into the following key areas. For detailed instructions, please refer to the linked documents.

### 1. Deployment and Installation

Proper deployment is the first step to leveraging the power of AuditToolkit. Our deployment guides cover everything from initial setup to advanced configuration.

- **[CMDB API Deployment & Configuration](./deployment/cmdb-api-deployment-and-configuration.md)**: Instructions for deploying and configuring the CMDB API data collection tool.
- **[Audit Toolkit Deployment Guide](./deployment/deployment-guide.md)**: A comprehensive guide to deploying the core Audit Admin Toolkit.
- **[Linux Security Lite Installation](./deployment/linux-security-lite-installation-guide.md)**: Step-by-step instructions for installing the Linux Security Lite expansion.

### 2. Licensing and Entitlement

Managing software licenses is critical for compliance and uninterrupted service.

- **[Licensing Model Overview](./licensing/licensing-model-overview.md)**: Understand the different licensing models available for the AuditToolkit platform.
- **[Offline Licensing Standardization](./platform/offline-licensing-standardisation-summary.md)**: Details on the process and benefits of standardizing offline licenses.

### 3. Security and Compliance

Security is at the heart of the AuditToolkit platform. These documents provide guidance on securing your installation and meeting compliance requirements.

- **[Secure Exposure Centre Hardening Guide](./docs/secure-exposure-centre-hardening-guide.md)**: Best practices for hardening the Secure Exposure Centre.
- **[Compliance Evidence Automation](./scripts/compliance-evidence-automation.md)**: Learn how to automate the generation of compliance evidence.
